Variables in Python

In this video, I have shown you What variable actually is and how to make them in Python

The word variable in programming describes a place to store information such as numbers, text, lists of numbers and text, and so on. Another way of looking at a variable is that it’s like a label for something.

For example, to create a variable named fred, we use an equal sign (=) and then tell Python what information the variable should be the label for. Here, we create the variable 'a' and tell Python that it labels the name "hello world" or anything else (note that this doesn’t mean that another variable can’t have the same value)

Syntax to make variable

First write name of the variable

There is many rules of naming the variable


Rules of naming the variable

  • Variables don't starts with numbers but you can use numbers between or end in a variable

  • Variables don't contain a special characters

  • Variables don't contain spaces (You can use underscore sign (-) instead of spaces)


Label the data that you want to store in that variable

For example:

a = "hello world"

Here we created the a variable label it with hello world

Printing variable

To print the variable

write print()

Write the name of the variable in the parenthese without qoutes

Rewriting the variable value

To rewrite the variable

First write the name of variable that you want to rewrite

Write equal sign (=) and name the variable

This will rewrite the variable

Like in this example :

a = "hello"

To rewrite it

a = "hello world"
