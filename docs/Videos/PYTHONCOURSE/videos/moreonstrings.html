Escape Sequences

In many programming languages including python and command line interfaces, the escape sequences are used in character literals (literals are basically corresponds to a short and easy way to specify a value) and string literals, to express characters which are not printable or clash with the syntax of characters or strings.

There are so many types of Escape Sequences in Python:-
Escape sequence Meaning
\’ This represents a single quote
\n This represents a newline
\r This represents a carriage return
\t This represents a tab
\b This represents a backspace
\f This represents a formfeed
\ooo This represents an octal value
\xhh This represents a hex value
\\ This represents a backslash
\uxxxx This represents 16 bits hex value
\uxxxxxxxx This represents 32 bits hex value
