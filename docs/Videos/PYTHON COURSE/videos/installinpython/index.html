Installing Python with Visual Studio Code and setting up VS Code to run Python

In This Video I have shown you how to install Python with Visual Studio Code

To install Python You have 2 things

1. Python Interpreter - which run our codes and gives output

2. IDE - Which give us a good envirment to write codes and decrease the efficient to our codes

In this course we will use Visual Studio Code as a IDE

Open Your Web Browser

Search for python.org

Go to the Official website of Python.org

Go to all releases under Downloads Tab

Download the latest version

But Note that the latest version which is 3.9 can only run on windows 8, windows 8.1, windows 10

If you are using Windows 7 You can download python version 3.8 which can run on all the operationg System

Now We have to download Visual Studio Code

To download VS code on your machine steps as follow

Search for Visual Studio Code

Go to https://code.visualstudio.com

Websit will automatically detcet what operating system you are using and simply click on Download

When all the Download wil complete you will see like this :

Now you have to install it

To install simply first run the python setup

Check add to path option, so the operating system will find where was python installed on your system

Now click on Install Now, Python Interpreter will install onto your system

Now Open VS Code Setup and install as follows

Check license agreement

Click on Next

Click on Next

Click on Next

Check these option as shown here

Click on Next

Click on Install

Your Python Interpreter and VS code has successfully Installed on your system

To Download simply click on extension in VS Code

Search for Python
