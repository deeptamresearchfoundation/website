Strings in Python

In this I have shown you what String actually is in Python

In programming terms, we usually call text a string. When you think of a string as a collection of letters, the term makes sense. All the letters, numbers, and symbols in this book could be a string. For that matter, your name could be a string, and so could your address. In fact, the first Python program we used a string: “Hello World.”

Strings are only in qotes that qoutes may be double qotes or single qotes

Syntax to make Strings

First write the name of the variable

And type = and under quotes type any text.

Like in this example

a = "Hello world"

print(a) #printing the a variable

print(type(a)) #To know the datatype of variable a


We get :

Hello world

class 'strings'


Concatanating string

if you want to concatanate a and b variable as follows

print(a + b) # concatanate variable a and b


Embedding Strings

To embed the string we use placeholders that is %s

Like in this example

# embedding strings

a = "I have Scored"

b = "1000 points"

print("I have Scored %s" % b) # %s is use to embed strings
